# 📚 Niji Seikei Skill Book Vol.07：例外処理（画像と掃除）

> [!TIP]
> **「ノイズ」を消して、「データ」だけを残す。** 断捨離の心や！

## 1. ヘッダー・フッターの掃除 🧹

PDFの各ページの上とか下にあるやつ。
*   `2024年度 就業規則` （右上にあるやつ）
*   `Confidential - 社外秘` （左下にあるやつ）
*   `- 12 -` （ページ番号）

これらは**「本文」ちゃうから、全部削除！** 🗑️
Markdownは「1つの繋がった巻物」みたいなもんやから、ページ区切りはいらんのよ。

## 2. 画像・図表の扱い 🖼️

PDFに貼ってある写真や複雑なグラフ。
これらはテキスト（文字）にできないから、基本は**「無視（スルー）」**や。

ただし、「ここに図があったこと」を残すために、以下のようにプレースホルダー（目印）を置くことが多いで。

*   **書き方**:
    ```text
    【図：組織図参照】
    ```
    ※もし図の中に「重要な文字情報」があって、それが本文に書かれてないなら、頑張って文字起こしすることもあるけど…基本は**「備考欄で相談」**が正解！勝手に判断しない！

## 3. 取り消し線（削除された条文） 📉

規程の改定履歴とかで、線で消されてる文章。
Markdownでは `~~` （波線2つ）で囲むと取り消し線になるで！

*   **書き方**:
    ```markdown
    ~~第3条（旧）~~ 削除
    ```
    → ~~第3条（旧）~~ 削除

これも「消すんか残すんか」迷ったら原本通りに残して、備考で聞く！
「迷ったら聞く」が最強のスキルやで！
