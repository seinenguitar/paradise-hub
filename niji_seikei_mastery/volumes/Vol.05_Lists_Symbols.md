# 📚 Niji Seikei Skill Book Vol.05：詳細ルール（記号とスペース）

> [!TIP]
> **「どこで区切るか」が勝負！** 記号の後の「全角・半角スペース」問題に終止符を打つで！

## 1. スペースを「入れない」パターン 🈲

「記号と文字がくっついてて当たり前」のやつら。
これらはワンセットで「ラベル」やから、引き離したらあかん！

*   **丸数字・カッコ付き数字**
    *   `①本件について` （OK）
    *   `① 本件について` （NG：離れすぎ！）
    *   `(1)実施事項` （OK）
*   **ピリオド付き数字**
    *   `1.はじめに` （OK）
*   **黒丸（リストマーカー）**
    *   `・本日は晴天なり` （OK）

## 2. スペースを「入れる」パターン 🆕

「番号」と「中身」をちょっと離した方が読みやすいやつら。
ここは**「半角スペース1つ」**を入れるんや！

*   **素の数字＋タイトル**
    *   `1 目的` （「1」と「目的」の間）
    *   `ア 定義`
*   **条番号＋タイトル**
    *   `第1条 目的` （OK：読みやすい！）
    *   `第1条目的` （NG：くっつきすぎ！）

### ⚠️ 例外：「カッコ」が続く時
直後に「（」が来る時は、スペースなしでOK！
*   `第1条（目的）` （OK：ひとかたまりに見える）

## 3. どっちか迷う「数量」系 🤔

文中で「1週間」とか出てきた時。これは「番号」ちゃう、「量」や！
だからスペースは入れない！

*   `1週間の猶予を与える` （OK）
*   `1 週間の猶予を与える` （NG：1. 週間の…って見えちゃう）

**【判断基準】**
*   **「項目番号」として使ってる？** → スペース入れる (`1 本文`)
*   **「言葉（単語）」として使ってる？** → スペース入れない (`1週間`)

迷ったら**「原本通り」**にして、備考欄で「どっちかわからんかった！」って正直に書くんやで。嘘つくより百倍マシや！
