## 1. アヤノ式・セルフチェックリスト ✅

これ全部にチェックがつかないと、納品したらあかん！

### 🔍 形式チェック
- [ ] **全角スペースは全滅させた？** （検索で `0件` になってる？）
- [ ] **画面プレビューは確認した？** （Markdownのコードだけ見て満足してない？）
- [ ] **テーブルは崩れてない？** （プレビューで枠線が綺麗に出てる？）

### 📖 内容チェック
- [ ] **文章は千切れてない？** （意味のない改行を撲滅した？）
- [ ] **条文の「第◯条」は見出しになってない？** （`#` つけてない？）
- [ ] **条文と条文の間は「1行」空いてる？**
- [ ] **リスト番号の後ろのスペースは正しい？** （Vol.05参照！）

## 2. よくあるミス（Before/After） 🙅‍♀️🙆‍♀️

| 項目 | ダメな例 ❌ | イケてる例 ⭕ |
| :--- | :--- | :--- |
| **条文** | `### 第1条` | `第1条` （平文で！） |
| **リスト** | `(1) 実施` | `(1)実施` （スペースなし！） |
| **表の改行** | （Enterキー押す） | `<br>` を使う！ |
| **不明点** | 勝手に直す | そのまま＋備考欄 |

## 3. 納品の儀作法 🎁

1.  **ファイル名**: 指示通りの名前になってる？（例：`社名_就業規則_md.md`）
2.  **備考欄**: 「ここ自信ないです」「原本が文字化けしてました」とか、正直に書く！
3.  **心構え**: 次の人がこのファイルを見た時、「おっ、こいつデキるな！」って思われる仕事をするんや！
